# 醉酒檢測遊戲 (Drunk Detection Game)

一個測試反應力和判斷力的單頁網頁遊戲。通過完成 10 個挑戰來證明你是清醒的！

## 🎮 遊戲玩法

- **目標**: 完成 10 個問題而不失去所有生命
- **生命**: 開始時有 3 條生命
- **時間限制**: 每題 5 秒鐘
- **挑戰類型**: 
  - 點擊最大顆按鈕
  - 點擊最小顆按鈕
  - 點擊數字最大的按鈕
  - 點擊數字最小的按鈕
  - 點擊指定數字的按鈕

## ✨ 特色功能

- 🎨 現代化深色主題設計
- 📱 響應式佈局（支援手機和桌面）
- ⏱️ 視覺化倒數計時器
- 🎯 隨機生成的按鈕（大小、顏色、形狀各異）
- 💫 流暢的動畫效果
- 🚀 無需安裝，開啟即玩

## 🚀 如何開始

### 方法 1: 直接開啟
1. 下載或克隆此儲存庫
2. 在瀏覽器中開啟 `index.html`
3. 點擊「開始」按鈕開始遊戲

### 方法 2: GitHub Pages
訪問線上版本：[遊戲連結]（待部署後更新）

## 🛠️ 技術規格

- **純 HTML/CSS/JavaScript** - 無框架依賴
- **單一檔案** - 所有程式碼包含在 `index.html` 中

- 現代網頁瀏覽器（Chrome 90+, Firefox 88+, Safari 14+, Edge 90+）
- JavaScript 已啟用

## 🎯 勝利條件

完成所有 10 題且生命值 > 0：顯示「恭喜你很清醒」

## 💔 失敗條件

生命值歸零：顯示「你醉了，別喝了」
